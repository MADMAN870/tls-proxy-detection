Proxy Setup

1. Download the CA certificate:

Download mitmproxy-ca.pem

2. Install the certificate as trusted:

  • Chrome: Settings → Privacy & Security → Security → Manage Certificates → Import → Trust for TLS
  • Firefox: Settings → Privacy & Security → Certificates → View Certificates → Import
  • Linux: sudo cp mitmproxy-ca.pem /usr/local/share/ca-certificates/ && sudo update-ca-certificates

3. Configure your browser proxy:

  • Set HTTP Proxy: {{ request.client.host if request.client else "your-docker-ip" }}
  • Set Port: 8082
  • Enable Use proxy for HTTPS
